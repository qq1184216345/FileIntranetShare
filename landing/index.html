局域网内的轻量文件 / 文本共享工具。桌面端启动服务，同网段的手机、平板、电脑扫码即可在浏览器收发文件与互传文本，无需注册也无需客户端。

下载 Windows x64 Linux 即将支持 查看所有版本 →

安装包约 2.79 MB · 运行内存 ~30 MB · 覆盖安装数据零迁移

核心特性

一个安装包解决局域网临时共享的所有场景

免安装访问

内嵌 HTTP + WebSocket 服务，手机扫二维码即开即用，访客端无需安装任何客户端。

大文件分片续传

浏览器端分片上传，断网重连接着传；单文件最大 10 GB，支持多文件并行队列。

密码保护

Argon2 哈希 + JWT 鉴权，改密码立即热生效，在线访客自动失效重新验证。

实时同步

任意端增删文件或文本，所有在线设备 WebSocket 即时刷新，永远看到同一份列表。

记录持久化

SQLite 存储分享记录，服务重启后下载链接依然可用；物理文件缺失自动 reconcile 清理。

托盘 · 暗色主题

最小化到系统托盘常驻；亮 / 暗 / 跟随系统三态切换，Guest 端亦跟随访客浏览器偏好。

三步上手

从安装到开始收发，通常不到一分钟

STEP 01

双击安装

NSIS 免管理员权限安装。首次启动服务时 Windows 弹一次防火墙入站请求，允许即可，后续不再提示。

STEP 02

扫码进入

主窗口显示局域网访问地址和二维码。同网段手机、平板、电脑扫码或直接打开浏览器输入网址即可进入访客页。

STEP 03

开始收发

拖文件进浏览器上传，点按钮下载，文本一键复制。主机端也可一键把本地文件 / 剪贴板加入分享列表。
